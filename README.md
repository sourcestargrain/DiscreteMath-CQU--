## 使用建议

1. **第一轮复习**：按 `05-teaching-notes/` 速成笔记过一遍知识点
2. **刷题巩固**：用 `03-textbook-solutions/` 逐题练习教材习题
3. **真题检验**：做 `01-past-exams/` 中的历年真题
4. **考前冲刺**：限时完成 `04-mock-exams/` 模拟卷，用 `06-concept-cards/` 快速回顾概念
